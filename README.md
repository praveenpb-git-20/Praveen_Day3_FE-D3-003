# FlowTrack 360 - Responsive Landing Page

## Project Overview

FlowTrack 360 is a modern, responsive landing page designed for workflow automation, project management, and team collaboration platforms. The website provides a professional user interface suitable for corporate organizations.

The project demonstrates the implementation of responsive web design principles using HTML, CSS, and JavaScript.

---

## Features

### Responsive Navigation Bar

* Company logo with Font Awesome icon
* Navigation links
* Mobile-friendly hamburger menu
* Sticky navigation bar

### Hero Section

* Professional business introduction
* Call-to-action buttons
* Dashboard preview card
* Two-column responsive layout

### Workflow Timeline

* Step-by-step process visualization
* Modern card-based design
* Responsive arrangement for mobile devices

### Statistics Section

* Business performance metrics
* Project, Client, and Employee statistics
* Clean and modern presentation

### Client Testimonial

* Customer feedback section
* Professional review card layout

### Footer

* Professional closing section

### Mobile Responsiveness

* Adaptive layouts for tablets and smartphones
* Responsive navigation menu

## How to Run the Project

### Method 1: Direct Browser Execution

1. Download the project files.
2. Place all files in the same folder.
3. Open `index.html` in any modern browser.

### Method 2: VS Code Live Server

1. Open project folder in Visual Studio Code.
2. Install the Live Server extension.
3. Right-click `index.html`.
4. Click **Open with Live Server**.
